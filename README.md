# Weather Free

A simple weather app built with Jetpack Compose.

[![All Contributors](https://img.shields.io/github/contributors/gongjieliu/freeWeather)](#contributors)

## Features
- View current weather and 5-day forecast
- Add/remove cities
- Support for multiple languages
- Temperature unit switching (Celsius/Fahrenheit)

## Tech Stack
- Kotlin
- Jetpack Compose
- Hilt (Dependency Injection)
- Room (Local Database)
- Retrofit (Networking)
- QWeather API

## Contributor
- **OpenCode** - AI-assisted development

## Build
```bash
./gradlew assembleDebug
```

APK will be at: `app/build/outputs/apk/debug/app-debug.apk`
